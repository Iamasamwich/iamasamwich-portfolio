Some of my projects:

Cycle Tracker

I made this app to track my fitness progression while cycling over time.

Once a user has created an account they can enter the details (date, time, distance, weight, and route details) of their bike rides, then view their aggregated data over time in graph form.

Cycle Tracker was styled with Semantic UI and uses Recharts for the graphs, and is hosted on Heroku with ClearDB as the MySQL host.

Note: Cycle Tracker was designed as a mobile app, so opening the app on a mobile phone will give the intended experience. On a desktop computer open the app in Chrome, then press Ctrl+Shift+J then Ctrl+Shift+M for mobile view.

Tic Tac Toe 3D

A 3 dimensional Tic Tac Toe game as a way to practice using React hooks.

This game uses three Tic Tac Toe boards stacked on top of each other so you can get winning combinations going down through the layers as well as on each layer.

With 3 boards there are now 49 winning combinations instead of the 8 on the traditional Tic Tac Toe.

Clock Dinger

This is a clock app, designed to be run in fullscreen mode and left on one of your workspaces.

This started off as a quick coding challenge, but now sits on my desktop permanantly to help me keep track of time.

By opening the burger menu you can select how often you want the chime to sound: never, or every 15, 30 or 60 minutes.
